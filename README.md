# BazaarBid

BazaarBid is a B2B wholesale bidding platform connecting Kirana store retailers with Big Market Suppliers.

## Setup

1. **Backend:**
   - cd `backend`
   - `npm install`
   - Set up `.env` based on `.env.example`
   - `npm run dev`

2. **Frontend:**
   - There are two portals: `retail-portal` and `seller-portal`.
   - cd into either and run `npm install` followed by `npm run dev`

## Features
- B2B Bulk Purchasing via reverse-auctions.
- Dynamic Subscriptions Plans.
- Anonymous Bidding mechanics.
